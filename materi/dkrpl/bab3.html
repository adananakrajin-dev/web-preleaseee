Bab 1: Struktur Dasar HTML5

HTML (HyperText Markup Language) adalah kerangka atau tulang punggung dari sebuah halaman web.

1.1 Apa itu Semantic HTML?

HTML modern menekankan penggunaan elemen semantik yang memiliki makna jelas bagi browser dan SEO.

  • <header>: Bagian kepala/navigasi.
  • <main>: Konten utama halaman.
  • <section>: Mengelompokkan konten berkaitan.
  • <article>: Konten mandiri (blog post).
  • <footer>: Bagian kaki halaman.

1.2 Anatomi Elemen HTML

<tagname class="nama-class">Konten yang muncul di web</tagname>

}
