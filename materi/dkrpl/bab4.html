OBJECT-ORIENTED PROGRAMMING // SEMESTER 2

Bab 1: Apa itu OOP?

OOP adalah paradigma pemrograman yang berorientasi kepada objek. Semua data dan fungsi dibungkus dalam kelas-kelas. Jika prosedural fokus pada langkah, OOP fokus pada interaksi antar objek.

Class

Cetak biru atau denah (blueprint).

Object

Bentuk nyata dari denah (Rumah).

Bab 2: Empat Pilar Utama

1. Encapsulation

Menyembunyikan data (Private).

2. Inheritance

Pewarisan sifat ke kelas anak.

3. Polymorphism

Satu objek, banyak bentuk.

4. Abstraction

Hanya menampilkan fungsi penting.

Bab 3: Implementasi C#

// Definisi Class
class Mobil {
  public string warna;
  public void Melaju() {
    Console.WriteLine("Mobil berjalan...");
  }
}

// Membuat Object (Instance)
Mobil mobilBaru = new Mobil();
mobilBaru.Melaju();
