Bab 1: Basis Data Relasional

Basis data relasional mengorganisir data ke dalam tabel yang saling berhubungan menggunakan kunci (keys).

Primary Key (PK)

Identitas unik tiap baris (Contoh: ID_Siswa).

Foreign Key (FK)

Penghubung antar tabel yang menunjuk ke PK.

Bab 2: Kategori Perintah SQL

DDL (Definition)
  • CREATE (Buat)
  • ALTER (Ubah Struktur)
  • DROP (Hapus Tabel)
DML (Manipulation)
  • INSERT (Tambah)
  • SELECT (Tampil)
  • UPDATE (Edit Data)

Bab 3: Implementasi Query

-- Membuat Tabel
CREATE TABLE siswa (
  id_siswa INT PRIMARY KEY,
  nama VARCHAR(100),
  jurusan VARCHAR(50)
);

-- Mengambil Data
SELECT * FROM siswa WHERE jurusan = 'RPL';
